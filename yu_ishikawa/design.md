# design.md

## 画面
1. ホーム画面  
   - タスク一覧を表示  
   - 新規タスク追加ボタン  
   - カテゴリフィルタ

2. タスク作成画面  
   - タスク名入力  
   - 締切日・優先度設定  
   - 保存ボタン

3. 設定画面  
   - テーマ変更（ライト/ダーク）  
   - 通知オン/オフ  
   - データバックアップ設定

---

## データ
1. タスク  
   - id, title, deadline, priority, category, status

2. カテゴリ  
   - id, name

3. 設定情報  
   - theme, notification, backup

---

## 処理の流れ
入力 → 確認 → 保存 → 表示

---

## 構成
画面（フロント） → API → サーバ → DB
