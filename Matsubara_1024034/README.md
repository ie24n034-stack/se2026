# 自作CLI学習管理ソフト

コマンドライン（CLI）から手軽にタスクを管理し、ポモドーロタイマーを使って学習時間を記録・可視化できる、受験生やエンジニア向けの学習管理ツールです。

## 主な機能
1. **ユーザー認証**: 新規アカウント登録、ログイン（パスワードはハッシュ化して安全に保存）
2. **タスク管理**: タスクの追加・一覧表示・完了・削除（優先度の設定やタスクIDの自動割り振りに対応）
3. **高機能タイマー**: 自由な時間指定、〇分〇秒フォーマット、一時停止（Stop）＆再開（Restart）に対応
4. **状況レポート**: 総学習回数・総学習時間の集計と、視覚的な目標達成率バーの表示

---

## 🛠️ 使い方とコマンド一覧

### 1. 準備（アカウント作成とログイン）
アプリを使用するには、まずアカウントを作成してログインします。

**新規アカウント作成**
python app.py register

**ログイン**
python app.py login

### 2. タスク管理
学習したい内容をタスクとして登録・管理します。

**タスクの追加**
python app.py add "基本情報の勉強" --priority 高

**未完了タスクの一覧表示**
python app.py list
# または
python app.py ls

**タスクの完了マーク**
python app.py done 1

**タスクの削除**
python app.py delete 1

### 3. タイマー機能

**タイマーの開始**
# 25分でスタートする場合
python app.py start 25m

# 45秒でスタートする場合
python app.py start 45s

# 時間を省略した場合は、テスト用に自動で15秒でスタートします
python app.py start

### 4. 学習レポートの確認

**レポートの表示**
python app.py summary
# または
python app.py stats
