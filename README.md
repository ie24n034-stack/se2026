# Simple Todo CLI 🚀

**ソフトウェア工学 演習プロジェクト**

日常のタスクをブラウザを開かずに、ターミナルから素早く管理するためのCLIツールです。

---

## ✨ 主な機能

- **シンプル操作**: `add`, `list`, `done` の最小限のコマンドで操作。
- **軽量・高速**: 依存関係を抑え、どんな環境でも瞬時に起動。
- **自動保存**: 実行ディレクトリ内の `tasks.json` にデータを永続化。

---
